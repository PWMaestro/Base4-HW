# Урок: строки
Строки и функции для работы со строками.

## Описание
Поставленые задачи:
* написать стандартные функции языка C++:
  1. strLen() - определение длины строки;
  2. strCpy() - копировани второй строки в первую переданную;
  3. strCat() - конкатенация строк (в первую переданную);
  4. strCmp() - сравнение строк.
* написать пользовательские функции:
  1. isLetter() - проверка на букву латинского алфавита;
  2. isWord() - провека на слово, состоящее из букв латинского алфавита;
  3. strToInt() - приведение числа-строки к целочисленному типу данных.

Реализация функций приведена в [файле](lesson-strings.cpp).
