# Урок: анализ строк
Разделение пользовательского текста на слова, выделяемые из текста разделителями, стоящими между слов.

## Описание
В ходе урока написана программа, которая проверяет, является ли символ в строке разделителем. Самостоятельно необходимо доработать задачу таким образом, чтобы программа выводила на экран все найденные в строке слова.

Реализация функций приведена в [файле](task-23-lesson.cpp).
