# Новобранцы

## Описание
> Новобранцы выстроены в шеренгу. По команде НАЛЕ–ВО! одни из них поворачиваются налево, а другие – направо. Это есть начальное положение шеренги. Далее каждый новобранец действует следующим образом: увидев перед собой лицо соседа, он ровно через секунду поворачивается кругом. В конце концов повороты прекратятся. Тогда сержант подает звуковой сигнал.
